# HitmanStatistics

A Hitman Contracts and Hitman Silent Assassin statistics tracker. 

# Features

***HitmanStatistics has following features***

- Shots fired
- Close encounters
- Headshots
- Alerts
- Kill count
- Harm count
- Innocent kill count
- Innocent harm count
- Analytics for Hitman Contracts
- Analytics for Hitman Silent Assassin
- Play time
